# 🚀 Sahil Somase Portfolio

A modern, responsive, and interactive personal portfolio showcasing my journey as an **AI & Machine Learning Engineer**. Built with React, Vite, Tailwind CSS, and Framer Motion, this portfolio highlights my technical skills, featured projects, education, achievements, and professional experience.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/SomaseSahil/Portfolio.git
```

### 2️⃣ Navigate to the Project

```bash
cd Portfolio
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Development Server

```bash
npm run dev
```

The application will start at:

```text
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
